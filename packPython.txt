cd el_GUI_comrado

Запуск PyInstaller:
    pyinstaller --onefile --windowed --name El_GUI_Comrado_v05 main.py


Есть несколько полезных опций:
Создание одного файла (--onefile): Упаковывает всё в один .exe-файл. Удобно для распространения, но может запускаться чуть дольше при первом старте.
Скрытие консольного окна (--windowed или --noconsole): Очень важно для GUI-приложений! Эта опция убирает черное консольное окно, которое появляется в фоне при запуске.
Название приложения (--name): Задает имя для вашего .exe-файла и папок сборки.
Иконка (--icon): Позволяет добавить иконку к вашему .exe-файлу (нужен файл .ico).
Рекомендуемая команда для вашего проекта:
Apply to main.py
Run
